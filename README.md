Preview the web application at: https://car-app-six-chi.vercel.app/

A modern car research and recommendation web app built with Next.js 16 + React 19, designed to help Indian buyers discover, compare, and shortlist cars tailored to their budget and lifestyle.

Features
Browse & Filter — Search across makes, body types, fuel types, transmission, price range, seating capacity, and safety rating
Car Detail Pages — Full specs, NCAP safety rating, similar car suggestions, and review scores
Side-by-Side Compare — Compare up to 3 cars simultaneously; best values highlighted in green
Personalised Quiz — 6-question quiz scores every car across budget, usage, family size, fuel preference, body style, and priorities
Shortlist — Save favourite cars with persistent localStorage storage
Instant Search — Debounced search by make, model, or variant (min 2 characters)
